História da Criação do HTML

No início da década de 1990, um cientista chamado Tim Berners-Lee trabalhava no CERN, um grande centro de pesquisas na Europa. Ele precisava de uma forma simples de compartilhar documentos entre pesquisadores que estavam em diferentes computadores.

Para resolver esse problema, ele criou uma linguagem simples chamada HTML, que significa HyperText Markup Language. Essa linguagem permite estruturar documentos utilizando elementos chamados de tags.
Essa linguagem se tornou a base da criação de páginas na internet.

Características do HTML

O HTML possui diversas características que facilitaram o crescimento da internet e o desenvolvimento de páginas web.

    Facil de aprender.
  • Baseado em Tags
  • Estrutura paginas na web
  • Funciona em qualquer navegador

Etapas básicas para criar uma página HTML

  1. Criar um arquivo com extensão .html
  2. Escrever a estrutura básica do documento
  3. Adicionar conteúdo usando tags
  4. Abrir o arquivo em um navegador

Desde sua criação, o HTML evoluiu bastante e hoje é uma das tecnologias fundamentais para o desenvolvimento da web.
